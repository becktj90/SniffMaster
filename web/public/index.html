Detection Pipeline

How raw signals become room-level classifications and scores.

Signal processing chain

I2C acquisition: the XIAO ESP32-S3 polls the Bosch BME688 over I2C for gas resistance, temperature, humidity, and pressure at the BSEC2 sample rate.

BSEC2 virtual sensors: Bosch's proprietary library derives IAQ (0–500), VOC equivalent (ppm), CO2 equivalent, stabilization status, and calibration accuracy from raw resistance patterns.

EMA smoothing: VOC readings pass through an exponential moving average filter to suppress single-sample noise while preserving rapid events like farts or cooking.

SmellNet classification: A TensorFlow Lite model trained with Bosch AI-Studio on labeled gas-response patterns scores 12 odor channels simultaneously (fart, cooking, alcohol, smoke, perfume, etc.).

Cross-correlation: Post-processor resolves misclassification by checking gasR patterns, CO2 correlation, humidity, and timing context against each odor hypothesis.

Home baseline: Device builds an EMA of quiet indoor readings over time. False positives are suppressed by comparing current readings against the learned home profile.

Composite scoring: Room Quality Score (0–100) fuses IAQ, VOC, CO2, humidity, odor detections, and outdoor AQI into a single actionable number.

Sensor Pipeline Pseudocode

Simplified logic flow for the main sensing loop running on the ESP32.

// ── SniffMaster Pro v4.0 Main Loop ──────────────────────

loop():
  handleButton()                    // gesture state machine
  bsecRan = envSensor.run()         // poll BME688 via BSEC2
  melTick()                         // background melody engine
  blePresenceTick()                 // BLE proximity scanner
  maintainWiFi()                    // multi-SSID failover
  pollPortalCommand()               // check for remote actions

  if newDataReady:
    // ── Raw → Derived ──
    vocSmooth = pushAndSmooth(vocRaw)     // EMA filter
    tier      = smellTier(vocSmooth)      // 0-5 severity
    scores[]  = scoreOdors(voc, iaq, co2, // SmellNet inference
                           temp, hum, gasR, dVoc)

    // ── Post-processing ──
    correctOdorScores(scores, gasR, co2)  // cross-correlation
    updateHomeBaseline(readings)          // learn quiet indoor
    applyHomeMlTuning(scores, baseline)   // suppress false pos
    deriveExpandedOdors(scores, context)  // 8 derived families
    stabilizePrimaryOdor(scores)          // 2-sample persistence

    // ── Composite assessment ──
    airScore = computeAirScore(iaq, voc, co2,
                               hum, scores, outdoorAqi)

    // ── Event detection ──
    if fartSpike(scores, dVoc, gasR):
      fartCount++; playAlert(); pushEvent()
    if dVocSpike or airScoreJump:
      playOdorChangeMelody(oldTier, newTier)

    // ── Cloud sync (gated by RSSI) ──
    if pendingAiFetch and rssi > -82:
      fetchGPTSassyMsg(snapshot)          // 5-15s HTTPS POST
    if webPostDue:
      sendToWebDashboard(snapshot)        // Vercel + Upstash

    // ── Display ──
    renderPage(displayPage)               // 7 auto-cycling pages
Signal Confidence Model
  • Direct measurement (high confidence): temperature (±1°C), humidity (±3% RH), pressure (±1 hPa), gas resistance (relative)
  • BSEC2 derived (medium-high): IAQ (calibration-dependent, accuracy 0–3), VOC equivalent, CO2 equivalent, stabilization status
  • ML classifier (medium): 12 odor channels scored 0–100%, cross-correlated with gasR/CO2/timing, 2-sample persistence filter
  • Composite scores (medium): Room Quality (0–100), CFI ventilation index, smell tier (0–5) — fuses multiple signals
  • AI commentary (low-medium): GPT-generated environmental analysis — entertaining but not a measurement
  • Experimental (low): paranormal scan, fart attribution, breath screening — novelty features

Applications
  • Indoor air quality: continuous IAQ/VOC/CO2 monitoring with trend analysis and ventilation recommendations
  • Occupancy sensing: CO2 accumulation rate and BLE proximity for meeting-room vitality assessment
  • HVAC diagnostics: room-to-room comparison of ventilation effectiveness using CFI and VTR indices
  • Odor classification: real-time 12-channel odor scoring with cross-correlation and home baseline suppression
  • Event detection: sub-second response to rapid VOC spikes (cooking, cleaning products, biological events)
  • Research platform: custom odor training via Bosch AI-Studio, labeled data export, ML model iteration
